# Additional resources

## [React.memo from the official React docs](https://reactjs.org/docs/react-api.html#reactmemo)

An API that can be used in conjunction with Context Providers to prevent unnecessary re-renders in top-level components in the tree. 

## [useMemo from the official React docs](https://reactjs.org/docs/hooks-reference.html#usememo)

The most popular open source form library for React. It saves you lots of time when building forms and offers a declarative, intuitive and adoptable paradigm.
